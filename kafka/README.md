# Origami Kafka

Документацию по работе можно найти в разделе [Origami Kafka](https://origamiframework.ru/docs.html#origami-kafka).

## Подключение

Необходимо добавить зависимость в pom.xml в проекте:
```XML
    <dependency>
        <groupId>ru.origamiframework</groupId>
        <artifactId>origami-framework-kafka</artifactId>
    </dependency>
```
