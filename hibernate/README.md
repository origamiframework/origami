# Origami Hibernate

Модуль для работы с базами данных на основе [Hibernate](https://hibernate.org/orm/documentation/6.1/).

Документацию по работе можно найти в разделе [Origami Hibernate](https://origamiframework.ru/docs.html#origami-hibernate).

## Подключение

Необходимо добавить зависимость в pom.xml в проекте:
```XML
    <dependency>
        <groupId>ru.origamiframework</groupId>
        <artifactId>origami-framework-hibernate</artifactId>
    </dependency>
```
