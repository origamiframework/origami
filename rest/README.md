# Origami RestAssured

Модуль для работы с HTTP-запросами на основе [RestAssured](https://rest-assured.io/).

Документацию по работе можно найти в разделе [Origami RestAssured](https://origamiframework.ru/docs.html#origami-restassured).

## Подключение

Необходимо добавить зависимость в pom.xml в проекте:
```XML
    <dependency>
        <groupId>ru.origamiframework</groupId>
        <artifactId>origami-framework-rest</artifactId>
    </dependency>
```
