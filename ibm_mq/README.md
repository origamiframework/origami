# Origami IBM MQ

Документацию по работе можно найти в разделе [Origami IBM MQ](https://origamiframework.ru/docs.html#origami-ibm-mq).

## Подключение

Необходимо добавить зависимость в pom.xml в проекте:
```XML
    <dependency>
        <groupId>ru.origamiframework</groupId>
        <artifactId>origami-framework-ibm-mq</artifactId>
    </dependency>
```
