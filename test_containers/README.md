# Origami Test Containers

Документацию по работе можно найти в разделе [Origami Test Containers](https://origamiframework.ru/docs.html#origami-test-containers).

## Подключение

Необходимо добавить зависимость в pom.xml в проекте:
```XML
    <dependency>
        <groupId>ru.origamiframework</groupId>
        <artifactId>origami-framework-test-containers</artifactId>
    </dependency>
```
