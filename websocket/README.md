# WebSocket

Документацию по работе можно найти в разделе [Origami WebSocket](https://origamiframework.ru/docs.html#origami-websocket).

## Подключение

Необходимо добавить зависимость в pom.xml в проекте:
```XML
    <dependency>
        <groupId>ru.origamiframework</groupId>
        <artifactId>origami-framework-websocket</artifactId>
    </dependency>
```
